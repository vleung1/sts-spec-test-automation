| File | Description |
|------|-------------|
| `icdc_enum_properties_summary.csv` | One row per property with an `Enum` block |
| `icdc_enum_terms_for_verification.csv` | One row per `(prop_handle, enum_value)` |
| `icdc_enum_terms_for_verification_enriched.csv` | Plus STS version, node, API **term_value** |
